# Neif Felder — Mechanical Engineer & Product Builder

Mechanical engineering student at Community College of Philadelphia (transfer track → Temple University, Fall 2027) with 8 years of hands-on automotive and fabrication experience. I build real things — CAD-designed aftermarket parts, performance vehicles, and now software tools for the automotive space.
